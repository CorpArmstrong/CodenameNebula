ТРЕБОВАНИЯ

### вершины модели
Все вершины модели должны находиться в пределах от -127 до +127 по X, Y и Z.

### Фейсы модели
все фейсы модели должны быть триангулированы. 
Конвертер поддерживает только треугольники.

### матерриалы и текстуры
На модели может быть какое угодно количество матерриалов.
но количество различных базовых текстур для этих матерриалов не должно превышать 8!
А имя каждого матерриала должно начинаться с имени текстуры и содержать аттрибуты(свойства) этого матерриала

таким образом каждая из восьми использующихся текстур может применться в разных местах с разными атрибутами

имена метерриалов должны строиться по вот таком принцыпу:

   texture_name.[Atr][Atr]...[Atr]

где вместо "texture_name" имя использующейся текстуры
а вместо "[Atr]" один или несколько свойтв этого матерриала
всего атрибутов семь некоторые из них могут комбинироваться.

атрибуты:
Ts - two sided (двустороннесть)
Tr - translucent (прозрачность)
Md - modulated (затемнённый)
Wp - weapon
Un - unlit     (матерриал на который не дествует освещение)
Fl - flat (обычный плоский)
En - Envmapped (зеркало)

например:

   windowGlassTexture.TrTs

   Этот матерриал используется для стекла какого нибуть окошка
   где "windowGlassTexture" - это текстура стекла этого окошка
   А "Tr" и "Ts" это аттрибуты этого матерриала 
   (Tr - translution делает материал просвещающимся)
   (Ts - translution делает матерриал видимым с обоих сторон как)
   
   windowFrameTexture. 

   Этот матерриал используется для рамки для какого нибуть окошка 
   где windowFrameTexture - это тектура этого окошка.
   А особенных аттрибутов(свойств) этот матерриал не имеет 
   потому вконце указана просто точка (.) без указания аттрибутов.
